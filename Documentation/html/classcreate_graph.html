Graph< int > createGraph::buildGraph ( projectData & data)
static

Transforms parsed ProjectData into a directed Flow Graph.

  • This function creates a source node, a sink node, and nodes for all reviewers and submissions. It adds edges with capacities based on expertise matching.
  • Parameters
    dataThe projectData structure containing all reviewers and papers.
    Returns
    A Graph object configured for the Max-Flow algorithm.
  • @time_complexity O(R * S) where R is the number of reviewers and S is the number of submissions, as we must check matches for every pair. @space_complexity O(V + E) where V = R + S + 2 and E is the number of valid matches.
