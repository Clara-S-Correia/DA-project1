bool Graph< T >::addFlowEdge ( const T & sourc,
const T & dest,
double cap )

Adds a directed flow edge between two vertices.

Creates both a forward edge with the given capacity and a reverse (residual) edge with 0 capacity. Links them via setReverse().

  • Parameters
    sourcThe info of the source vertex.
    destThe info of the destination vertex.
    capThe maximum capacity of the flow edge.
    Returns
    true if both vertices exist and the edges were created, false otherwise.
  • @time_complexity O(V) due to findVertex calls (scanning the vertex set).
